System Features

Role-Based Access

Different dashboards for Students, Faculty, HODs, Deans, and Principal with appropriate permissions.

Academic Management

Comprehensive tools for attendance, grading, lesson planning, and academic monitoring.

Centralized Data

All academic and administrative data stored in a centralized database for easy access and reporting.

About the Project

The GCEK ERP System is being developed as a centralized solution to streamline academic and administrative operations in Government College of Engineering Kalahandi (GCEK).

The system provides role-based access for multiple stakeholders to enhance efficiency, transparency, and management. Built with modern web technologies including HTML, CSS, Bootstrap, JavaScript, Spring Boot, and MySQL.
