The GCEK ERP System is a centralized web-based Enterprise Resource Planning (ERP) platform developed for Government College of Engineering Kalahandi (GCEK).
The project aims to digitize and streamline academic and administrative operations by integrating students, faculty, departmental heads, and administrative authorities into a single unified system.

The system follows a Role-Based Access Control (RBAC) approach to ensure security, transparency, and efficient management across the institution.
